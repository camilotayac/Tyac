# Branding: Identidad visual y Logo

Tyac no es solo un nombre; es una identidad visual construida para transmitir claridad y modernidad. En esta guía veremos cómo el diseño tipográfico y los colores de marca se unen para crear el branding de la plataforma.

## 1. El Logo "Tyac"

En lugar de usar una imagen pesada, el logo de Tyac es texto puro con estilos CSS avanzados. Esto garantiza que se vea nítido en cualquier resolución y cargue de forma instantánea.

### Estilo Tipográfico:
Usamos una fuente **Sans-Serif** de peso extremo (`font-weight: 700`) con un espaciado entre letras negativo (`letter-spacing: -0.5px`). Este estilo es típico de las marcas tecnológicas modernas (como Meta o Apple).

```css
.header-logo {
    color: var(--clr-white-soft);
    font-size: 30px;
    font-weight: 700;
    letter-spacing: -0.5px;
    transition: color 0.2s ease;
}
```

## 2. Paleta de Acento

El branding de Tyac se apoya en el color **Púrpura Dracula** (`#bd93f9`). Este color se asocia tradicionalmente con la creatividad y el conocimiento, pero en su tono neón también evoca el mundo del código y el desarrollo.

## 3. Estados de Interacción

El logo reacciona al usuario. Al pasar el cursor, cambia sutilmente de blanco al púrpura de la marca. Es una micro-interacción que confirma que el logo es también el botón de "Inicio".

## 4. El Favicon Vectorial

Para el icono de la pestaña del navegador, usamos un **SVG**. A diferencia de los antiguos archivos `.ico`, los SVG son escalables y pueden cambiar de color según el modo del sistema operativo (oscuro/claro) del usuario.

> [!TIP]
> **Consistencia**: El nombre "Tyac" aparece siempre con la misma tipografía y espaciado tanto en el Navbar como en el Footer, reforzando el reconocimiento de marca en toda la experiencia.
