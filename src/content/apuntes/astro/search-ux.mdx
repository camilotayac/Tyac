# Búsqueda Híbrida: Hero vs Header

El sistema de búsqueda de Tyac es uno de sus pilares tecnológicos. No se trata simplemente de una caja de texto; es una experiencia **híbrida** diseñada para estar disponible exactamente cuando el usuario la necesita, adaptando su comportamiento según el contexto.

## 1. El concepto de Ubicuidad

En Tyac, la búsqueda está en dos lugares clave:
- **Hero (Página de Inicio)**: Una barra grande y central enfocada en la exploración inicial ("¿Qué quieres aprender hoy?").
- **Header (Persistente)**: Una barra compacta en la navegación que acompaña al usuario en cada lección.

## 2. Diferenciación Funcional

A pesar de ser el mismo componente (`Buscador.astro`), Tyac les da comportamientos distintos:

### El Buscador del Hero (Trigger)
No realiza búsquedas locales. Actúa como un **disparador (trigger)** que abre el Modal de Búsqueda Global. Esto permite que el usuario tenga una experiencia inmersiva y enfocada en una interfaz más grande.

### El Buscador del Header (Live Search)
Realiza búsquedas en tiempo real mostrando un **dropdown** de resultados inmediatos. Es ideal para saltar rápidamente entre temas mientras se está estudiando.

## 3. Experiencia de Usuario (UX)

Hemos diseñado la búsqueda para que se sienta instantánea. Usando JavaScript moderno, los resultados aparecen al escribir apenas dos caracteres, eliminando la fricción y la necesidad de navegar a una página de resultados separada.

> [!TIP]
> **Atajo Senior**: En Tyac, puedes abrir el buscador global desde cualquier lugar presionando `Cmd + K` (o `Ctrl + K`). Es un estándar de las mejores herramientas de desarrollo que hemos traído a esta plataforma.
