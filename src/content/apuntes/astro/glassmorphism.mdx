# Glassmorphism en Tyac

Si Tyac se siente "moderno" y "profundo", es en gran parte gracias a nuestra implementación del **Glassmorphism**. Esta técnica visual imita la apariencia del cristal esmerilado, permitiendo que el contenido flote sobre el fondo de una manera elegante y ligera.

## 1. Los 3 Ingredientes del Cristal

Para lograr el efecto que ves en nuestras tarjetas y en el Modal de Búsqueda, combinamos tres propiedades de CSS:

### A. Transparencia (Background)
Usamos colores en formato `rgba` para que el fondo del elemento deje pasar un poco de luz.
```css
background: rgba(255, 255, 255, 0.03);
```

### B. Desenfoque (Backdrop Filter)
Esta es la clave. Difumina lo que está detrás del elemento, creando la sensación de cristal.
```css
backdrop-filter: blur(12px);
```

### C. Borde de Contorno (Border)
Un borde muy fino y semi-transparente que simula el grosor del cristal y captura los "reflejos" del borde.

## 2. Por qué lo usamos en Tyac?

En interfaces oscuras (Dark Mode), los elementos negros sobre fondo negro se pierden. El Glassmorphism nos permite crear **jerarquía visual** sin necesidad de usar colores de fondo sólidos que harían la web más pesada visualmente.

## 3. Implementación en el Modal de Búsqueda

El sistema de búsqueda global usa un Glassmorphism extremo para enfocar la atención del usuario en el input de búsqueda mientras mantiene el contexto de la página anterior de forma borrosa en el fondo.

```css
.search-modal-container {
    background: rgba(10, 11, 16, 0.8);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.1);
}
```

## 4. Rendimiento y Compatibilidad

El `backdrop-filter` es una propiedad costosa para el navegador. En Tyac, la usamos con moderación y aseguramos que haya un color de fondo sólido de respaldo (fallback) para navegadores antiguos que no soporten este efecto.

> [!IMPORTANT]
> **Detalle Tyac**: Al aplicar este efecto en el Navbar, logramos que el contenido de la página "brille" a través de la barra mientras el usuario hace scroll, creando una experiencia mucho más orgánica.
