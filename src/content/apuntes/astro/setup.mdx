# Configuración Inicial

Para empezar a desarrollar en Tyac, primero necesitamos preparar nuestro entorno local.

## Requisitos Previos

- **Node.js**: Versión 18.14.1 o superior.
- **NPM** o **PNPM**: Para gestionar las dependencias.
- **VS Code**: Recomendamos instalar la extensión oficial de Astro.

## Instalación

1.  Clona el repositorio: `git clone https://github.com/camilotayac/Tyac`
2.  Instala las dependencias: `npm install`
3.  Inicia el servidor de desarrollo: `npm run dev`

¡Listo! El sitio estará disponible en `localhost:4321`.
