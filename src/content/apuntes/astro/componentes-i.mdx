# Componentes Atómicos I: TarjetaHorizontal

En Tyac, la **Tarjeta Horizontal** es el elemento que más se repite en el Index. Fue diseñada para ser compacta pero rica en información, facilitando que el usuario escanee rápidamente los cursos y apuntes disponibles.

## 1. Diseño de Información

Cada tarjeta muestra cuatro datos clave:
1.  **Icono**: Una representación visual de la materia.
2.  **Título**: El nombre del curso o guía.
3.  **Descripción**: Un resumen rápido de lo que aprenderás.
4.  **Meta-info**: El número de temas o lecciones.

## 2. Anatomía de las Props

Para que el componente sea reutilizable, definimos una interfaz estricta. Esto asegura que siempre que usemos una tarjeta en Tyac, le pasemos exactamente lo que necesita:

```astro
---
interface Props {
  titulo: string;
  descripcion: string;
  icono?: string;
  cantidadTemas: string;
}
const { titulo, descripcion, icono = "📄", cantidadTemas } = Astro.props;
---
```

## 3. Estilos Premium con Glassmorphism

Las tarjetas de Tyac usan un efecto de "cristal" para integrarse con el fondo oscuro. Usamos un borde semi-transparente y un ligero cambio de color al pasar el cursor:

```css
.tarjeta-cuerpo {
    background: rgba(255, 255, 255, 0.03);
    border: 1px solid rgba(255, 255, 255, 0.08);
    transition: all 0.3s cubic-bezier(0.16, 1, 0.3, 1);
}

.tarjeta-cuerpo:hover {
    background: rgba(255, 255, 255, 0.06);
    border-color: var(--dracula-purple);
    transform: translateX(8px); /* Sutil desplazamiento a la derecha */
}
```

## 4. Estructura de "Slot" Implícito

Aunque este componente no usa `<slot />` (porque todo se pasa por props), su diseño permite que el `titulo` y la `descripcion` se adapten al espacio disponible mediante el uso de `line-clamp` en CSS, evitando que el texto largo rompa la cuadrícula.

> [!TIP]
> **Consistencia de Marca**: Al usar el color `--dracula-purple` en el borde al hacer hover, reforzamos la identidad visual de Tyac en cada interacción del usuario.
