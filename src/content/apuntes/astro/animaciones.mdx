# Animaciones reveal-up y decoraciones

Un sitio web premium se diferencia por los detalles. En Tyac, no queríamos que los elementos simplemente "aparecieran"; queríamos una entrada elegante que guiara la vista del usuario. Para lograrlo, implementamos un sistema de micro-animaciones y elementos decorativos.

## 1. El sistema `reveal-up`

Esta es nuestra animación principal. Se encarga de hacer que el texto suba desde abajo mientras gana opacidad. 

### El código CSS:
```css
@keyframes reveal-up {
    from {
        opacity: 0;
        transform: translateY(30px);
        filter: blur(10px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
        filter: blur(0);
    }
}
```

Lo aplicamos con escalonamiento (**delays**) para que el título aparezca primero, luego el subtítulo y finalmente el buscador. Esto crea un dinamismo rítmico.

## 2. Elementos Decorativos Flotantes

Alrededor del Hero de Tyac, verás emojis como 🧬, ⚛️ y 🚀. No son imágenes, sino texto dentro de capas con animaciones de **órbita** y **flotación**.

### Técnica de "Órbita" con CSS:
Usamos transformaciones compuestas para que los elementos giren en un eje imaginario:
```css
@keyframes orbit {
    from { transform: rotate(0deg) translateX(50px) rotate(0deg); }
    to { transform: rotate(360deg) translateX(50px) rotate(-360deg); }
}
```

## 3. Interactividad Sutil

Además de las animaciones de entrada, el buscador del Hero tiene un efecto de **levitación (lift)** al pasar el cursor. Esto le indica al usuario que el elemento es interactivo sin necesidad de ponerle un botón gigante que diga "Haz clic aquí".

> [!TIP]
> **Menos es Más**: En Tyac usamos una opacidad del `0.2` para los elementos flotantes. Así decoran el espacio sin distraer de la lectura del contenido principal.
