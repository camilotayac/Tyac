# Course Landing: 00-LandingLayout

Cuando un usuario hace clic en un curso desde el inicio, llega a la página de presentación. En Tyac, esta página es gestionada por el `00-LandingLayout.astro`. Su misión es vender el curso al estudiante: mostrarle qué aprenderá, qué dificultades tiene y quién lo imparte.

## 1. El diseño de "Header de Curso"

Este layout presenta un encabezado imponente con un fondo desenfocado que hereda el color de la materia. Si el curso es de Química, el aura será rojiza; si es de Astro, será púrpura.

## 2. Metadatos de un Vistazo

Implementamos una barra de metadatos dinámica que muestra:
- **Dificultad**: Representada con nuestro sistema de estrellas.
- **Lecciones**: El conteo total de temas.
- **Categoría**: La materia a la que pertenece.

## 3. El Grid de Acceso Directo

El layout divide la pantalla en una sección de información (izquierda) y una vista previa del temario (derecha). Esto permite que el estudiante vea todo el contenido antes de decidir inscribirse o empezar la primera clase.

## 4. Dinamismo de Props

El `00-LandingLayout` recibe el objeto `curso` completo. Esto significa que con un solo archivo de CSS y estructura, podemos mostrar cualquier materia de Tyac con solo cambiar los datos de entrada.

```astro
---
// LandingLayout.astro
const { curso } = Astro.props;
---
<div class="landing-header">
    <h1>{curso.titulo}</h1>
    <p>{curso.descripcion}</p>
</div>
```

> [!IMPORTANT]
> **Toque Pro**: Hemos diseñado este layout para que incluya etiquetas SEO específicas del curso, ayudando a que cada materia particular sea indexable de forma independiente por los motores de búsqueda.
