# El Orquestador de Vistas: Layouts

En Tyac, los layouts no solo envuelven el contenido, sino que inyectan lógica de navegación.

## LayoutBase vs LayoutLessons

- **LayoutBase**: Se encarga del Head, Meta tags globales y el Header estático.
- **LayoutLessons**: Gestiona el Sidebar, los Breadcrumbs, la Paginación y el área de lectura Markdown.
