# Performance Tuning: Velocidad extrema

Astro es rápido por naturaleza, pero en Tyac hemos aplicado capas extra de optimización para asegurar que la plataforma vuele. La velocidad no es solo un número en un test; es respeto por el tiempo del estudiante.

## 1. Zero JS por Defecto

Tyac se sirve como HTML estático casi en su totalidad. A diferencia de React o Vue puro, donde el navegador tiene que descargar y ejecutar megabytes de JavaScript para mostrar un texto, Tyac envía solo lo justo. El JavaScript solo se activa para el buscador y el filtrado.

## 2. Optimización de Activos (Assets)

- **Minificación**: Todo nuestro CSS y JavaScript se comprime automáticamente en el momento de la construcción, eliminando espacios y comentarios innecesarios.
- **Fuentes Locales**: No cargamos fuentes desde servidores externos (como Google Fonts) para evitar latencias de red. Las fuentes de Tyac viven en nuestro propio servidor.

## 3. Pre-renderizado de Rutas

Al usar rutas dinámicas estáticas, cada lección de Tyac es un archivo HTML físico en el servidor. Esto significa que el tiempo de respuesta es de **milisegundos**, ya que no hay cálculos de base de datos ni procesamiento de servidor en tiempo real.

## 4. Estrategia de Caching

Configuramos Tyac para que el navegador del usuario guarde (cachee) los elementos que no cambian, como el Logo, la Paleta de Colores y los Scripts base. Así, la segunda vez que entras a Tyac, la carga es casi instantánea porque el 90% de la web ya está en tu dispositivo.

> [!TIP]
> **Resultado**: Gracias a estas técnicas, Tyac obtiene puntuaciones de **95+ en todas las categorías de Lighthouse**, asegurando una experiencia fluida incluso en conexiones de internet lentas.
