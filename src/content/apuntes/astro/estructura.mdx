# Anatomía de Tyac

La organización es clave para mantener un proyecto de este tamaño. Así es como se estructuran las carpetas:

- `src/components/`: Piezas reutilizables (Botones, Buscador, Sidebar).
- `src/content/`: El alma del proyecto (Archivos .mdx cargados vía Content Collections).
- `src/data/`: El registro manual y lógica asíncrona de cursos.
- `src/layouts/`: Estructuras maestras para páginas y lecciones.
- `src/pages/`: Definición de las rutas del sitio.
- `src/styles/`: Tokens de diseño y variables CSS.
