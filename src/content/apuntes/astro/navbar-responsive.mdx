# Mobile First: Navbar responsiva

Un gran porcentaje de estudiantes accede a Tyac desde sus teléfonos. Por eso, el Navbar debe ser flexible y adaptarse sin perder funcionalidad ni legibilidad.

## 1. El desafío del espacio

En una pantalla de 375px, no podemos tener el Logo, el Buscador y los Enlaces en una sola fila. En Tyac, priorizamos los elementos según el ancho de la pantalla:

### Estrategia en Tyac:
1.  **Desktop**: `Logo | Buscador | Enlaces`.
2.  **Tablet**: `Logo | Buscador` (Enlaces se mueven a un menú o se simplifican).
3.  **Mobile**: `Logo (Izquierda) | Buscador (Reducido)`.

## 2. Flex Wrap y Flex Grow

Usamos las propiedades de Flexbox para que los elementos se "empujen" o se expandan según sea necesario. El buscador tiene un `flex: 1` para ocupar todo el espacio disponible entre el logo y los enlaces.

```css
.logo-container {
    display: flex;
    align-items: center;
    gap: 20px;
    flex: 1; /* Esto hace que el contenedor del buscador crezca */
}
```

## 3. Media Queries de Interrupción

En Tyac usamos puntos de ruptura claros. A partir de los **900px**, el diseño se vuelve más espacioso y los enlaces de navegación ganan protagonismo. Por debajo de esta medida, simplificamos la UI para evitar el "amontonamiento" de texto.

## 4. El buscador auto-adaptable

El componente `Buscador.astro` cambia su ancho automáticamente. En móviles, se vuelve más estrecho pero mantiene su funcionalidad de filtrado, permitiendo que el usuario encuentre lecciones rápidamente sin importar el dispositivo.

> [!IMPORTANT]
> **Toque Final**: Hemos asegurado que los botones de navegación tengan un área táctil de al menos 44px, cumpliendo con los estándares internacionales de accesibilidad móvil (a11y).
