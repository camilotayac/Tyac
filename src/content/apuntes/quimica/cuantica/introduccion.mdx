# El Átomo y la Mecánica Cuántica

La química cuántica aplica la mecánica cuántica a problemas químicos.

## Dualidad Onda-Partícula

Los electrones no son solo pequeñas esferas; se comportan como ondas y partículas al mismo tiempo.

### La Ecuación de Schrödinger

Esta ecuación describe cómo cambia el estado cuántico de un sistema físico con el tiempo. Es la base para entender los orbitales atómicos.
